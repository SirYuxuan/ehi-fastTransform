## 简介
Ehi-FastTransform 目前提供了如下功能
- 转换为英文
- 转换为驼峰
- 转换为下划线
- 转换为大写
- 转换为小写
- 智能变量名推荐(选中中文,按下 alt + shift + e )
### 快捷键
- 一键转换 alt + shift + w
- 智能变量名推荐 alt + shift + e
## 安装
在IntelliJ IDEA中依次打开

`File->Setting->Plugin->Marketplace`

输入 `FastTransform` 进行搜索
